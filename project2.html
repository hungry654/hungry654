🥈 2번 프로젝트 - VR 기반 감각형 게임 기획 『DreamLink』

📌 프로젝트 개요

  • 프로젝트명: DreamLink (예정)
  • 장르: VR 기반 감각형 인터랙티브 어드벤처
  • 테마: 시간 제한 속에서 “오감(五感)”이 순차적으로 사라지는 미션 클리어 게임
  • 핵심 키워드: #VRImmersion #감각제한 #미션타이머 #상호작용게임

📖 기획 배경

  • 기존 VR 게임들이 ‘추격·탐험·퍼즐’ 위주였다면, 오감 제한 → 미션 달성 → 시간 연장의 반복 구조로 새로움 제공
  • 감각 상실 상태별로 달라지는환경 반응과 상호작용에 집중

🧠 시스템 특징

기능 설명
오감 제한 모드 일정 간격으로 시각·청각·촉각·후각·미각 중 하나씩 랜덤 비활성화 → 몰입도 극대화
미션 타이머 제한 시간 내 지정된 미션(수수께끼 해결·아이템 찾기 등)을 완료해야만 다음 감각 유지 가능
상호작용형 퍼즐 제한된 감각을 보완하는 VR 전용 인터페이스(햅틱 컨트롤러, 공간 음향)로 문제 해결
동적 환경 적응 플레이어 상태(어느 감각이 사라졌는지)에 따라 AI가 실시간 환경 효과(빛·소리·진동)를 조정

🧙 예시 에피소드

  • 챕터 1: 소리 없는 숲
    청각이 사라진 숲속에서발소리 진동 피드백과 시각 큐로 길 찾기
  • 챕터 2: 어둠의 동굴
    시각을 잃은 동굴에서공간 음향과 손끝 햅틱만으로 미로 탈출
  • 챕터 3: 냄새 없는 시장
    후각을 잃은 시장에서 단서가 되는시각 기호·촉각 텍스처 수집 미션

📈 향후 계획

  • Unity & WebXR 기반 프로토타입 완성
  • 뇌파(EEG) 센서 연동해 ‘정서 반응’ 실시간 피드백 시스템 개발
  • 사용자 데이터 기반 AI 추천 미션 → 퍼스널라이즈드 경험 제공
  • 크라우드펀딩 → VR 아케이드·교육용 버전 출시

💬 사용 기술 (예정)

  • Unity & C# (VR 렌더링·인터랙션)
  • WebXR & Three.js (웹 배포용 경량 버전)
  • Python (AI 백엔드) - 환경·미션 난이도 동적 조정
  • EEG 모듈 (생체 신호 연동 프로토타입)
