💬 3번 프로젝트 - 웹 기반 감성 챗봇 서비스 제작『마음꽃』

📌 프로젝트 개요

  • 프로젝트명: 『마음꽃』(예정)
  • 기획 테마: AI 기반 감성 챗봇 서비스 기획

📖 기획 개요

  • 복잡한 설명 없이 한 문장 키워드만 입력해도,
    사용자의 감정 상태에 맞춘공감형 대화와 위로 메시지를 제공하는 웹 챗봇.
  • 하루의 스트레스, 외로움, 불안 등 간단한 감정을 타이핑하면,
    AI가 “엄청 힘들었죠? 잘 견뎌냈어요.” 같은 따뜻한 답변.

🧠 시스템 특징

기능 설명
✅ 감정 키워드 입력 “불안해”, “속상해” 등 간단 키워드 입력만으로도 대화 시작 가능
🤗 공감형 응답 GPT API 기반으로 사용자의 감정을 진단해 맞춤형 위로·격려 메시지 제공
💬 대화 기록 보관 이전 대화 이력을 보관함에 자동보관. 사용자가 언제든 다시 읽을 수 있도록 함.
🌱 힐링 큐레이션 대화 중 긍정 문구·음악·짧은 명언 등 힐링 콘텐츠 링크 제공
📊 감정 로그 시각화 하루·주간 단위로 입력된 감정 키워드 빈도를 차트로 확인

🖼️ UI/UX 콘셉트

  • ✨ 따뜻한 파스텔 톤 버튼과 말풍선 디자인
  • 💬 말풍선은손글씨 스타일 폰트사용, 편안한 대화 느낌 강조
  • 🖼️ 심플한 채팅 인터페이스, 불필요한 요소 배제
  • 📈 감정 로그 차트는막대 그래프나 워드 클라우드로 귀엽게 표현

💬 사용 기술 (예정)

  • JavaScript & React: 인터랙티브 챗 UI 구현
  • HTML5 / CSS3 (Tailwind): 감성적이고 반응형 디자인
  • OpenAI GPT API: 감정 분석·공감형 응답 생성
  • LocalStorage: 대화 이력 임시 저장
